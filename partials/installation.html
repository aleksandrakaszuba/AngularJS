Angular JS installation - steps

1. Get AngularJs

Go to: https://angularjs.org/ and select needed version and additional modules. Not all are included in the base version of angularJs (ie. ng-routes)

Aditional modules:

angular-route.js: Adds routing support. Most popular!.

angular-animate.js: Useful for creating nice CSS3 animations.

angular-mocks.js & angular-scenarios.js: This is useful for testing purposes.

2. Include AngularJs

Include angular script in your index.html file

Normally, calls to JavaScript frameworks are placed at the bottom of the page. That way, the layout loads first, and then application would load next. But Angular JS is doing a lot more than just adding basic functionality to your HTML. It's sort of going to take control of all your layout. So it should go at the very top, so it loads it first.

1. Include by adding a script tag with a path to the downloaded script

2. Addtional modules should be added below


<script src="js/angular.js"></script>
<script src="js/angular-route.js"></script>

or add a link to a copy hosted on CDN


<!doctype html>
<html ng-app>
<head>
	<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.6/angular.min.js"></script>
</head>
