Angular JS installation - steps

1. Download AngularJs

Download from: https://angularjs.org/

2. Include AngularJs

Normally, calls to JavaScript frameworks are placed at the bottom of the page. That way, the layout loads first, and then application would load next. But Angular JS is doing a lot more than just adding basic functionality to your HTML. It's sort of going to take control of all your layout. So it should go at the very top, so it loads it first.

<!doctype html> <html ng-app> <head> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.6/angular.min.js"></script> </head>
