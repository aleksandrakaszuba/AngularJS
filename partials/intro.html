Introduction to Angular JS

What is AngularJS?

  1. Open-source, front-end JavaScript framework
  2. Originally, developed by Misko Hevery and Adam Abrons in 2009
  3. Now, maintained by Google
  4. Single Page Application(SPA)

Features

  1. Powerful JavaScript based development framework
  2. Clien side application in a clean Model View Controller(MVC) way
  3. Cross-browser compliant
  4. Open source, completely free

Technological bases

Strength

  1. Data Binding
  2. Customize and Extensible
  3. Support
  4. Compatibility
  5. Testing

Weakness

  1. SEO problem in AngularJS applications
  2. Easy to start, hard to master
  3. AngularJS implementations scale poorly
  4. The libraries have to be AngularJS specific
  5. Does not work if user disables the JavaScript in browser
